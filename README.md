**RigWrap Plugin by** `@oohypr`
	
**Description:**  
RigWrap allows you to add Classic Shirts, Classic Pants and Shirt Graphics to R6 and R15 Rigs without Humanoids.

**Dev Forum Post:**  
https://devforum.roblox.com/t/3751017

**Creator Store:**  
https://create.roblox.com/store/asset/71774011419996

**Benchmark Video:**  
https://devforum-uploads.s3.dualstack.us-east-2.amazonaws.com/uploads/original/5X/5/6/e/d/56ed956f38135437d68c766880688c6a604866eb.mp4

**RESULTS:**  
- Much better performance when rendering many NPCs
- Massively reduced NPC loading time

**LIMITS:**  
- Rendering does not work properly for the following Avatar Types:  

	- Mesh2012 (R6 ONLY)  
	-  Mesh2016 (R6 ONLY)  
	-  Rthro (R6 ONLY)
